# React Todo App

A modern Todo application built with **React** and **TypeScript**. The app allows users to create, edit, complete, delete, and filter tasks with a clean and responsive interface.

## Live Demo

🔗 [Demo](https://palfinger2002.github.io/react-todo-app/)

## Repository

🔗 [GitHub Repository](https://github.com/Palfinger2002/react-todo-app)

---

## Technologies

- React
- TypeScript
- HTML5
- SCSS
- Vite
- ESLint
- Prettier

---

## Features

- Add new todos
- Edit existing todos
- Delete single todo
- Delete all completed todos
- Toggle todo completion
- Toggle all todos
- Active / Completed / All filters
- Items left counter
- Persistent storage (LocalStorage)
- Responsive design
- Error handling
- Loading indicators

---

## Getting Started

### Clone the repository

```bash
git clone https://github.com/Palfinger2002/react-todo-app.git
```

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm start
```

or

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
